# OS-Assignments

This repository contains the code and report for the "Computer Operating System" comprehensive design project.

## 项目简介
该项目是HLJU计算机操作系统课程设计，包含以下四个部分：

1. 进程控制
2. 分页式存储管理
3. 文件与磁盘管理
4. 进程调度

## 环境依赖

- **编程语言**: Python 3.9
- **软件包依赖**:
  - pandas==2.2.2
  - tabulate==0.9.0
  - numpy==1.23.0
